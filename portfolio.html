Python Snake Game

Aufbau

Python ist der Kopf meines Spiels, die Gesamte Logik wird von ihm übernommen. Alles was mit Rechnen oder oder Inputs und Outputs zu tun wird mithilfe von Python erledigt. Tkinter ist mein GUI(Graphical User Interface). Es hat mir ermöglicht, Figuren zu erstellen, ein Map zu zeichnen und Farben hinzufügen. Grob gesagt, habe ich mit Tkinter meinen Python Code zum Leben erweckt.

Gameplay

Das Gameplay folgt jeder klassische Snake Spiel. Man muss so viel wie möglich essen um Punkte zu sammeln und länger werden zu kénnen. Gleichzeitig muss man dararauf achten, nicht gegen die wände oder sicht selbst zu stossen, sonst verliert man! Das Spiel ist am Anfang ist das Spiel langsam aber später werden mehr Nahrungsmittel auf die Map erscheinen um den Fortschritt zu beschleunigen

Mobile Game

Aufbau

Thunkable, ein Online-Tool ist was ich benutzt habe um den Mobile-App zu erstellen. Mit der GUI von Thunkable habe ich mehrere Seiten erstellt und knopfe hinzugefügt. Die wichtigste Seite aber ist die GameScreen, wo der Gameply stattfindet. Dort habe ich viel Logic Blöcke benutzt um den Logic des Spiels zu erstellen. Denken Sie wie Scratch aber komplizierter.

Gameplay

Man muss die blaue Plattform ziehen und den Ball ins gegnerische Tor schiessen, während man sein eigenes Tor verteidigt. Der Gegner ist autonom und schiesst zurück und blockiert die Schüsse. Wer zuerst zehn Tore erreicht, gewinnt.

Unerwartete Hindernisse

Ursprünglich hatte ich geplant, das Spiel zum Testen bereitzustellen, aber Thunkable bietet nur sehr eingeschränkte Funktionen und verlangt gleichzeitig ein teures Abonnement. Ausserdem ist die Leistung sehr instabil, weshalb ich das Spiel nur durch Bilder und Videos präsentieren kann.

Python Calculator

Aufbau

Python ist der Gehirn meines Taschenrechners. Der gesamte Code ist damit geschrieben. Die Gesamte Input geht hindrin, alle Operation werden dadrin ausgeführt und die Gesamte Output kommt von ihn. Tkinter ist die Bibliothek die ich benutzt habe um die GUI(Graphical User Interface) zu erstellen. Zuerst habe ich ein Fenster ersellt, indem ich Knopfe hinzugefügt habe. Danach hab ich jeder Knopf ein Funktion gegeben und zum Schluss habe ich alles scchén formatieret.

Verwendung

Mit dem kann man viele Operatinen ausführen, nämlich 1-Addition, 2-Subtraktion, 3-Multiplikation, 4-Division, 5-Potenzierung, 6-Wurzelziehen und 7-Prozentrechnen.
