Python Snake Game

Aufbau

Python ist der Kopf meines Spiels, die gesamte Logik wird von ihm übernommen. Alles was mit Rechnen oder oder Inputs und Outputs zu tun hat, wird mit Hilfe von Python erledigt. Tkinter ist mein GUI(Graphical User Interface). Es hat mir ermöglicht, Figuren zu erstellen, eine Map zu zeichnen und Farben hinzuzufügen. Grob gesagt, habe ich mit Tkinter meinen Python Code zum Leben erweckt.

Gameplay

Das Gameplay folgt jedem klassischen Snake Spiel. Man muss so viel wie möglich essen, um Punkte zu sammeln und länger zu werden. Gleichzeitig muss man dararauf achten, nicht gegen die Wände oder sich selbst zu stossen, sonst wird man verlieren! Das Spiel ist am Anfang langsam aber später werden mehrere Nahrungsmittel auf der Karte erscheinen, um den Fortschritt zu beschleunigen.

Mobile Game

Aufbau

Das Online-Tool Thunkable habe ich benutzt, um die Mobile-App zu erstellen. Mit der GUI von Thunkable habe ich mehrere Seiten erstellt und Knöpfe hinzugefügt, um das ganze App zu gestalten Der wichtigste Teil aber ist der GameScreen, wo das Gameplay stattfindet. Dort habe ich viele Logic Blöcke , um die Logik des Spiels zu erstellen. Denken Sie wie Scratch aber komplizierter.

Gameplay

Man muss die blaue Plattform ziehen und den Ball ins gegnerische Tor zu schiessen, während man sein eigenes Tor verteidigt. Der Gegner ist autonom und schiesst zurück und blockiert die Schüsse. Wer zuerst zehn Tore erreicht, gewinnt.

Unerwartete Hindernisse

Ursprünglich hatte ich geplant, das Spiel zum Testen bereitzustellen, aber Thunkable bietet nur sehr eingeschränkte Funktionen und verlangt gleichzeitig ein teures Abonnement. Ausserdem ist die Leistung sehr instabil, weshalb ich das Spiel nur durch Bilder und Videos präsentieren kann.

Python Taschenrechner

Aufbau

Python ist das Gehirn meines Taschenrechners. Der gesamte Code ist damit geschrieben. Der gesamte Input geht hinein, alle Operation werden darin ausgeführt und der gesamte Output kommt von ihm. Tkinter ist die Bibliothek, die ich benutzt habe, um die GUI(Graphical User Interface) zu erstellen. Zuerst habe ich ein Fenster erstellt, indem ich Knöpfe hinzugefügt habe. Danach habe ich jedem Knopf eine Funktion gegeben und zum Schluss habe ich alles schön formatiert.

Verwendung

Mit dem Taschenrechner kann man viele Operatinen ausführen, nämlich:

  • Addition
  • Subtraktion
  • Multiplikation
  • Division
  • Potenzierung
  • Wurzelziehen
  • Prozentrechnen
